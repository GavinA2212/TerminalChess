## Setup

### Prerequisites
- [.NET 9.0 SDK](https://dotnet.microsoft.com/download) or later

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/GavinA2212/TerminalChess.git
   cd TerminalChess
   ```

2. **Restore dependencies**
   ```bash
   dotnet restore
   ```

3. **Build the project**
   ```bash
   dotnet build
   ```

4. **Run the application**
   ```bash
   dotnet run
   ```
